How to use:
1.) adapt the jobdict.txt to change the neuron numbers, simtime etc.
2.) adapt neurondict.txt to change neuron parameters
3.) adapt gif2_gain.sh to change number of data points and simultaneous processes.
The number of data points must be changed in lines 3 and 11, it is just denoted as a number.
The number of data points should be odd.
4.) type sbatch gif2_gain.sh
